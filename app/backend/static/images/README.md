# Image Assets

## Required: hero-texture.jpg

Please place the newspaper texture image as `hero-texture.jpg` in this directory.

The CSS is configured to:
- Display at 15% opacity
- Use multiply blend mode for transparency
- Apply a radial gradient mask that fades from center to edges
- Position it centered behind the hero headline

**Expected dimensions:** 1200x800px or higher resolution
**Format:** JPG (optimized for web)
